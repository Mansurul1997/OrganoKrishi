# OrganoKrishi

React + Tailwind frontend with a Node.js Express backend.

## Setup

1. Open a terminal in `d:\Projects\Mansurul\Mansurul\small project`
2. Run `npm install`
3. Run `npm run dev`

Frontend: http://localhost:5173
Backend: http://localhost:4000

## Structure

- `client/` - Vite React + Tailwind frontend
- `server/` - Express API server with product data
